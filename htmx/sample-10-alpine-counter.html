10) Alpine.js カウンター

Reactより軽く、HTML中心で書けるリアクティブな最小例。

デモ（実動）

現在値: 0

保存先キー: demo.alpine.counter

解説

  • x-data で状態とメソッドを定義
  • x-text で表示をバインド
  • @click でイベント処理
  • 更新時に localStorage へ保存

このサンプルのコード

<section x-data="counter()" x-init="init()">
  <strong x-text="count">0</strong>
  <button @click="inc">+1</button>
</section>

function counter() {
  return {
    count: 0,
    init() { this.count = Number(localStorage.getItem('demo.alpine.counter') ?? 0) },
    inc() { this.count++; localStorage.setItem('demo.alpine.counter', this.count) }
  }
}

補足（運用時）

このページでは分かりやすさ優先でCDNを使用。運用環境ではローカル配置（self-host）推奨。
